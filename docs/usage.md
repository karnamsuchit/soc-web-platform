# Usage Guide

## Demo Monitoring Workflow

1. Start the backend and frontend.
2. Open the dashboard.
3. Use the `Monitoring` tab to view the simulated live SOC feed.
4. Open the `Alerts` tab to review generated detections.
5. Open the `Log Analysis` tab to inspect normalized event evidence.
6. Open the `IOCs` tab to review extracted indicators.

## Analyst Workflow

- Start with high and critical alerts.
- Review source IP, affected URL/path, and evidence.
- Check MITRE ATT&CK technique mapping.
- Use the log viewer to validate the alert context.
- Use the IOC view to identify suspicious sources and affected URLs.

## Built-In Datasets

The platform includes preloaded datasets for SSH brute force, SQL injection, XSS, reconnaissance, suspicious HTTP requests, and Apache access logs.
